\# ◆ CLOUD EXPLANATION



Cloud = renting computing power instead of owning it.



Analogy:

\- Owning a car vs using Uber



\---



\# ◆ REGIONS EXPLANATION



Explain:

AWS has global locations called Regions.



Use analogy:

\- Countries = Regions



\---



\# ◆ AZ EXPLANATION



Availability Zones = multiple buildings in same city.



If one fails:

others continue working.



\---



\# ◆ KEY TEACHING MOMENT



Say:



“Cloud is not one server — it is a distributed system.”
