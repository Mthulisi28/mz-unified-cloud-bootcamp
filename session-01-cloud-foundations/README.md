\---



\# ◆ 1. WHAT IS CLOUD COMPUTING?



Cloud computing = On-demand access to computing resources over the internet.



Instead of owning servers, companies:

\- Rent computing power

\- Scale resources instantly

\- Pay only for usage



\---



\# ◆ 2. WHY CLOUD COMPUTING?



Benefits:

\- Cost efficiency

\- Global reach

\- High availability

\- Elastic scaling

\- Faster innovation



\---



\# ◆ 3. AWS GLOBAL INFRASTRUCTURE



AWS operates globally using:



\- Regions

\- Availability Zones

\- Edge Locations



\---



\# ◆ 4. REGIONS



A Region is a geographical area (e.g. London, Cape Town, Frankfurt).



Each Region contains multiple Availability Zones.



\---



\# ◆ 5. AVAILABILITY ZONES (AZs)



AZs are physically separate data centers within a Region.



They provide:

\- Fault tolerance

\- High availability

\- Disaster recovery



\---



\# ◆ 6. KEY PRINCIPLE



“AWS is global, but your applications must be designed for resilience.”
